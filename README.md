Как подключить прокси
Откройте сайт на любом устройстве
Нажмите кнопку «Подключить в Telegram» под любым прокси
Telegram откроется и предложит подтвердить подключение — нажмите «Подключиться»
Если Telegram заблокирован и не открывается:

Установите его через APK с официального сайта (Android)
Или через VPN на 1 раз — только для установки
Как это устроено
text

каждый час
     │
     ▼
GitHub Actions (сервер вне РФ)
     │
     ├── парсит 3 Telegram-канала с прокси
     │       ProxyMTProto · MTProtoProxies · mtproto_proxy
     │
     ├── TCP-проверяет каждый прокси параллельно (~3 сек на всё)
     │       живые → сортирует по задержке (ms)
     │       мёртвые → убирает с главной, оставляет в архиве
     │
     ├── хранит историю 7 дней в data/proxies.json
     │
     └── генерирует docs/index.html и docs/archive.html
              └── публикует на GitHub Pages
Стек: Python 3.11 · requests · BeautifulSoup4 · GitHub Actions · GitHub Pages

Структура репозитория
text

update_proxies.py          # единый скрипт: парсинг + проверка + кеш + HTML

docs/
├── index.html             # главная страница (свежие живые)

Запустить вручную
Если нужны прокси прямо сейчас, не дожидаясь следующего часа:

Actions → Auto Update Proxies Website → Run workflow

Страница обновится примерно через 1 минуту.

Про TCP-проверку
Зелёный значок ✅ и задержка в мс означают что порт отвечает на TCP-соединение. Это не 100% гарантия работы MTProto — итоговую проверку делает сам Telegram в момент подключения. На практике ~90% зелёных прокси работают.

Хотите форкнуть?
Нажмите Fork в правом верхнем углу
Зайдите в Settings → Pages → Source → выберите ветку main, папку /docs
Разрешите Actions: Settings → Actions → Allow all actions
Запустите вручную: Actions → Run workflow
Готово — ваша личная копия с автообновлением.
